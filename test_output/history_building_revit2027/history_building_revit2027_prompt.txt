这张图是一张**历史建筑图版 / 古典建筑设计图**，上半部分是**建筑正立面**，下半部分是**首层平面图**。文字是法语，整体风格像 18 世纪法国建筑书里的住宅或府邸设计范例。

### 1. 图纸内容

上半部分标题大意是：

> **一个建筑的展开图：正面宽十八托瓦兹，进深十一托瓦兹**

这里的 **toise / 托瓦兹** 是法国旧制长度单位。

图的上方展示的是建筑的**正立面**，可以看到：

- 中央是突出入口门廊；
- 有一排古典柱式；
- 正中顶部有半圆形或穹顶式屋顶构件；
- 左右两翼对称；
- 两侧都有带弧形山花的端部体量；
- 立面强调轴线、比例和对称性。

这是一种很典型的**法国古典主义 / 早期新古典主义住宅立面**。

------

### 2. 下半部分是首层平面

下面写着：

> **Plan du Rez-de-Chaussée**

意思是：

> **首层平面图 / 一层平面图**

平面图也是严格对称的。中间是主要礼仪空间，左右两边是不同功能房间。

可以看到几个法语房间名称：

| 法语                   | 含义              |
| ---------------------- | ----------------- |
| **Péristyle**          | 柱廊 / 门廊空间   |
| **Sallon / Salon**     | 客厅 / 沙龙       |
| **Salle d’Assemblée**  | 集会厅 / 会客厅   |
| **Salle de compagnie** | 社交厅 / 陪客厅   |
| **Salle à manger**     | 餐厅              |
| **Cabinet**            | 书房 / 小会客室   |
| **Chambre à Coucher**  | 卧室              |
| **Antichambre**        | 前厅 / 等候室     |
| **Garderobe**          | 衣帽间 / 更衣室   |
| **Office**             | 备餐间 / 服务空间 |
| **Billard**            | 台球室            |
| **Côté du Jardin**     | 花园侧            |
| **Côté de la Cour**    | 庭院侧            |

所以它不是现代住宅户型图，而更像是一个**贵族住宅、府邸、乡间别墅或 hôtel particulier 的设计范例**。

------

### 3. 空间组织特点

这张图最重要的特点是：**强轴线、强对称、礼仪性很强**。

中轴线大概是：

**庭院侧入口 → 半圆形台阶 → 柱廊 Péristyle → 中央 Salon → 花园侧弧形平台 / 台阶 → 花园**

也就是说，人从庭院进入建筑，经过柱廊，再进入中央大厅/沙龙，最后面向花园。这是典型的法式古典住宅布局。

左右两侧则布置功能房间：

- 左侧有卧室、前厅、书房、集会厅；
- 右侧有餐厅、台球室、社交厅、书房；
- 中央 Salon 是最重要的公共空间。

------

### 4. 建筑风格描述

这张图可以描述为：

> 一张 18 世纪法国古典主义建筑图版，展示了一座对称府邸的正立面与首层平面。建筑采用中央轴线布局，正立面由柱廊、山花、檐口和对称翼部构成，体现出强烈的秩序感和礼仪性。平面中庭院侧和花园侧通过中央门廊与沙龙串联，左右两翼分布卧室、书房、餐厅、会客厅和服务空间，反映出法国贵族住宅中公共接待、私人居住和服务功能的分区关系。

一句话总结：
**这是法国古典主义府邸/别墅的建筑图版，上面是立面，下面是首层平面，重点展示了对称轴线、中央沙龙、庭院与花园之间的礼仪性空间组织。**
