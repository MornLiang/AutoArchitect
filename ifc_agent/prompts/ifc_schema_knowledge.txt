## IFC Entity Type Hierarchy

### Spatial Structure (containment hierarchy)
- IfcProject → IfcSite → IfcBuilding → IfcBuildingStorey → IfcSpace
- Each level "decomposes" into the next via IfcRelAggregates
- Building elements are "contained in" spatial elements via IfcRelContainedInSpatialStructure

### Common Building Elements
- **Walls**: IfcWall, IfcWallStandardCase (straight walls with constant cross-section)
- **Slabs**: IfcSlab (floors, ceilings, roof slabs; predefined types: FLOOR, ROOF, LANDING, BASESLAB)
- **Roofs**: IfcRoof (container for roof slabs, predefined types: FLAT_ROOF, SHED_ROOF, GABLE_ROOF, HIP_ROOF)
- **Doors**: IfcDoor (with IfcDoorType for reusable definitions)
- **Windows**: IfcWindow (with IfcWindowType)
- **Columns**: IfcColumn (vertical structural members)
- **Beams**: IfcBeam (horizontal/inclined structural members)
- **Stairs**: IfcStair, IfcStairFlight, IfcRailing
- **Openings**: IfcOpeningElement (voids in walls/slabs for doors/windows)
- **Spaces**: IfcSpace (functional areas/rooms within a storey)
- **Furniture**: IfcFurniture, IfcFurnishingElement

### Material Assignments
- IfcRelAssociatesMaterial links products to materials
- Material representations: IfcMaterial, IfcMaterialLayer, IfcMaterialLayerSet, IfcMaterialLayerSetUsage
- Walls typically use IfcMaterialLayerSetUsage (multiple layers: insulation, structure, finish)

### Common Property Sets (Pset)
- **Pset_WallCommon**: IsExternal, LoadBearing, ThermalTransmittance, Reference
- **Pset_SlabCommon**: IsExternal, LoadBearing, ThermalTransmittance
- **Pset_DoorCommon**: IsExternal, FireRating, Reference
- **Pset_WindowCommon**: IsExternal, ThermalTransmittance, GlazingAreaFraction
- **Pset_SpaceCommon**: IsExternal, GrossPlannedArea, NetPlannedArea
- **Pset_BuildingStoreyCommon**: AboveGround, EntranceLevel

### Quantity Sets (Qto)
- **Qto_WallBaseQuantities**: Length, Width (thickness), Height, GrossVolume, NetVolume, GrossSideArea, NetSideArea
- **Qto_SlabBaseQuantities**: Width (thickness), Perimeter, GrossArea, NetArea, GrossVolume
- **Qto_DoorBaseQuantities**: Width, Height, Area
- **Qto_WindowBaseQuantities**: Width, Height, Area
- **Qto_SpaceBaseQuantities**: GrossFloorArea, NetFloorArea, Height, GrossVolume

## Building Analysis Rules

### Structural System Identification
- If IfcColumn count is high with a regular grid → Frame structure
- If IfcWall is primary with wall thickness ≥ 200mm and few columns → Load-bearing wall structure
- Mixed systems: both significant columns and load-bearing walls

### Enclosure Analysis
- Fully enclosed: all boundary walls are IfcWall (not IfcRailing or IfcCurtainWall)
- Open areas: IfcRailing or gaps in wall boundaries indicate semi-outdoor spaces

### Ventilation Compliance
- Each habitable IfcSpace should have at least one IfcWindow on an exterior wall
- Window-to-floor area ratio typically ≥ 1:7 for natural ventilation compliance

### IFC Type Classification Issues
- Roof components classified as IfcBuildingElementProxy or generic models indicate misclassification
- Elements with PredefinedType "NOTDEFINED" may need review
